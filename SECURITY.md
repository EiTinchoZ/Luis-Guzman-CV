# Security Policy

## Supported Versions

Only the `main` branch is supported.

## Reporting a Vulnerability

If you discover a security issue related to this portfolio project, report it privately before disclosing it publicly.

- Primary contact: `luiseudoro@gmail.com`

Please include:

- a clear description of the issue
- steps to reproduce it
- expected impact
- any suggested remediation

Acknowledgement target: within 72 hours.
